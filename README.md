# Spam Email Detection Project

## Project Snapshot
This project helps identify whether a message is Spam or Not Spam using Machine Learning.

## Problem Statement
Many users receive unwanted promotional or harmful messages.  
The goal of this project is to automatically classify such messages and separate them from normal messages.

## Project Idea
The model learns from example messages and then predicts the class of a new message.

## How It Works
- Messages are collected into a dataset
- Text is converted into numeric form using TF-IDF
- A Machine Learning model is trained on the data
- The model predicts whether a message is Spam or Not Spam

## Technologies Used
- Python
- Google Colab
- Pandas
- Scikit-Learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Output
Input:
`Congratulations! You have won a cash prize`

Prediction:
`Spam`

## Key Learnings
- Text classification
- Feature extraction
- Model training
- Prediction on new messages

## Future Scope

- Build a simple web app for real-time detection
